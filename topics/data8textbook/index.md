The textbook **Computational and Inferential Thinking: The Foundations of Data Science, 2nd Edition**, is available at <https://inferentialthinking.com/>, and is the main textbook developed for the [Data 8 course at UC Berkeley](https://data8.org/). 

When the textbook is used at UC Berkeley, they use the `datahub.berkeley.edu` computing environment, which already has access set up to many of the assets in the textbook (e.g. dependencies, datafiles, etc.).

When using it at UCSB or anywhere else, it is very helpful to know where to find all of these dependencies.  The key is knowing to look in the *repository for the textbook* which can be found here:
* <https://github.com/data-8/textbook>

In particular, these items are helpful to be able to find:
